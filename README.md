# Resume Analyzer + JD Matcher

## 📌 Overview
A full-stack web application that analyzes a resume against a job description and provides:
- Skill matching
- Missing skills
- Match percentage
- Improvement suggestions

This project demonstrates frontend-backend integration using React and FastAPI.

---

## 🚀 Features
- Extracts skills from resume text
- Compares resume with job description
- Calculates match score
- Identifies missing skills
- Provides improvement suggestions

---

## 🛠 Tech Stack
- Frontend: React
- Backend: FastAPI (Python)
- API: REST APIs

### Frontend
cd frontend

npm run dev
